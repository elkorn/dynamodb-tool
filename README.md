## Getting Started

Install the module with: `npm install db-scanner`

```js
var db-scanner = require('db-scanner');
db-scanner.awesome(); // "awesome"
```

Install with cli command

```sh
$ npm install -g db-scanner
$ db-scanner --help
$ db-scanner --version
```
